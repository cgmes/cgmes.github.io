All synchronous machine detailed types use a subset of the same data parameters and input/output variables.  <br /></p><p>The several variations differ in the following ways:<br /></p><p><ul><li>The number of  equivalent windings that are included</li><br /></p><p><li>The way in which saturation is incorporated into the model.</li><br /></p><p><li>Whether or not “subtransient saliency” (<b>X''q</b> not = <b>X''d</b>) is represented.</li><br /></p><p></ul><b>Note:</b> It is not necessary for each simulation tool to have separate models for each of the model types.  The same model can often be used for several types by alternative logic within the model.  Also, differences in saturation representation may not result in significant model performance differences so model substitutions are often acceptable. <br /></p>

The Subtransient Salient Pole Type is modelled using one of the concrete child classes of the SynchronousMachineDetailed class.    If the SynchronousMachineTimeConstantReactance class is used, .modelType = subtransient and .rotorKind = salientPole.<br /></p><p>The d-axis equivalent circuit is the same as for the Subtransient Round Rotor Type. The q-axis has only one equivalent rotor winding, which may be labelled as transient (<b>X'q</b>) or subtransient (<b>X''q</b>); <b>X'q</b> is used for the CIM description.<br /></p><p><font color="#0f0f0f">Ifd represents LadIfd (XadIfd) and is a voltage proportional to field current in the per unit system ( IEEE 421.5-2005, Annex B).  It is equal to Efd in steady-state.  </font><br /></p><p><b><br /></p><p></b><b>Parameter Notes:</b><br /></p><p><ol><li><b>X''q</b> (=<b>X'q</b>) is assumed to be equal to <b>X''d</b> (no subtransient saliency).</li><br /></p><p><li>Saturation is modelled in the d axis only. </li><br /></p><p><li>For generators, the following input parameters may be omitted with default shown: <b>X'q</b> (=<b>X''d</b>), <b>X''q</b> (=<b>X''d</b>), <b>T''qo</b>, <b>Ks</b> (=0), <b>S1q</b> (=0), <b>S12q</b> (=0), <b>Tc</b> (=0).</li><br /></p><p><li>For motors, the following input parameters are not used:  <b>X'q</b>, <b>X''q</b>, <b>T''qo</b>.</li><br /></p><p></ol><br /></p>

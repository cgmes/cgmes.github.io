The Subtransient Type J Type is modelled using one of the concrete child classes of the SynchronousMachineDetailed class.  If the SynchronousMachineTimeConstantReactance class is used, .modelType = subtransientTypeJ.<br /></p><p>This model is the WECC type J model with uniform inductance ratios rotor modelling with modified saturation model; shaft speed effects are neglected. This model is the same as Subtransient Type F but includes the effect of generator loading on saturation.  With the usage of the saturation constant (<b>Ks</b>) parameter, it allows reproduction of the measured "V curves" over the whole range of reactive power, particularly on some hydro (salient pole) machines.<br /></p><p>Saturation is modelled in both the d and q axes.<br /></p>

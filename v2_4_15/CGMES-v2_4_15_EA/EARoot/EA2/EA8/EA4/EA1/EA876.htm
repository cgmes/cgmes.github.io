The simplified model represents a synchronous generator as a constant internal voltage behind an impedance (<b>Rs</b> + <b>jXp</b>) as shown in the Simplified diagram.<br /></p><p><br /></p><p>Since internal voltage is held constant, there is no <b>Efd</b> input and any excitation system model will be ignored.  There is also no <b>Ifd</b> output.<br /></p><p><br /></p><p>This model should not be used for representing a real generator except, perhaps, small generators whose response is insignificant.  <br /></p><p><br /></p><p>The parameters used for the Simplified model include:<br /></p><p><ul><li>RotatingMachineDynamics.damping (D)</li><br /></p><p><li>RotatingMachineDynamics.inertia (H)</li><br /></p><p><li>RotatingMachineDynamics.statorLeakageReactance (used to exchange jXp for SynchronousMachineSimplified)</li><br /></p><p><li>RotatingMachineDynamics.statorResistance (Rs).</li><br /></p><p></ul><br /></p>

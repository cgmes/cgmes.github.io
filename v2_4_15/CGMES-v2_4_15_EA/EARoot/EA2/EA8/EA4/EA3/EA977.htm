This diagram illustrates the inputs and outputs for a turbine-governor.<br /></p><p><b><br /></p><p></b><b>Notes: </b><br /></p><p><ol><li><b>Pref</b> is usually held constant for stability analysis, but may be determined by a user-defined model or, for long-term dynamics, an area-wide automatic generation control (AGC) model.  For some models, <b>Pref</b> has units of PU speed instead of PU power.</li><br /></p><p><li>Some turbine-governor models are used in cross-compound interconnections, e.g. GovSteam1, where they are connected to two generators (synchronous machines). </li><br /></p><p><li>The vast majority of turbine-governor models have mwbase as a parameter.  This parameter corresponds to GeneratingUnit.ratedGrossMaxP from the Production package, but is specified explicitly in the Dynamics package to support use of a different value by transient analysis tools, if needed.</li><br /></p><p></ol><br /></p>

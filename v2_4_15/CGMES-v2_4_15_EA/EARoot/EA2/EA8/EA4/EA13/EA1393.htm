The class represents IEEE PF Controller Type 2 which is a summing point type controller and makes up the outside loop of a two-loop system. This controller is implemented as a slow PI type controller. The voltage regulator forms the inner loop and is implemented as a fast controller.<br /></p><p><br /></p><p>Reference: IEEE Standard 421.5-2005 Section 11.4. <br /></p>

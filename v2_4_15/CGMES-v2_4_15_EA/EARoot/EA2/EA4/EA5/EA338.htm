Equipment.aggregate provides an alternative way of representing an aggregated (equivalent) element by allowing usage of all available attributes for a given class <b>only if</b> usage of dedicated classes for equivalent equipment is not possible due to limited number of attributes on these classes. In this case and if the flag is set to "true", the equipment is treated as an equivalent obtained by a network reduction procedure. <br /></p><p>The attribute is not used for the following classes: PowerTransformerEnd, BusBarSection, EquivalentBranch, EquivalentShunt and EquivalentInjection.<br /></p>
